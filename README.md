# Disk Eject

The **one and only** purpose of Disk Eject is to eject all external disk(s) from macOS.

## Install

```
go get github.com/jaux/diskeject
```

## Run

```
$GOPATH/bin/diskeject
```

Or, if you have added `$GOPATH/bin` to your `$PATH`, you can simply run

```
diskeject
```
